# Streamlit
> Naver Boostcamp leve1 competition "Mask Classification" model serving 실습

## Installation
```
git clone 
```

## Tech/Framework Used
- `Python 3.11.5`

__built with__
- `VSCODE`

## Requirements
```
Pillow==10.0.1
streamlit==1.29.0
torch==2.1.2
torchvision==0.16.0
```

## File
- mask.py : mask classification model inference 과정을 streamlit으로 구현
- mask_child.jpg : 예시 사진
- best.pth : model train 결과

## Run
-`streamlit run mask.py`
